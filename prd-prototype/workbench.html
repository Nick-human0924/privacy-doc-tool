隐私不是一句口号

页面需要让用户知道文件去了哪里，也要知道哪里还不能做。

文件不离开浏览器

当前静态版本没有上传接口，文件通过 File API 读入浏览器内存，处理完成后由浏览器生成下载。

涂黑不是盖一层遮罩

可视化涂黑会先渲染页面，再写入黑块并栅格化输出，避免原文字层被复制出来。

能力边界直接展示
